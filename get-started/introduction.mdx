---
title: What is Autorepo?
description: "Autorepo is an AI software engineer that you can interact with through GitHub issues."
---

## Features

<CardGroup cols={2}>
  <Card
    title="AutoRefine"
    icon="microscope"
    href="/features/autorefine">
    Automatically refine issues by adding context from the codebase, online documention, and by working with the user to clarify any ambiguities.
  </Card>
  <Card
    title="AutoPlan"
    icon="list-check"
    href="/features/autoplan">
    Automatically create a step-by-step plans to complete issues by performing extensive research throughout the codebase and across the web.
  </Card>
  <Card
    title="AutoPR"
    icon="code-pull-request"
    href="/features/autopr">
    Automatically write pull requests for new issues, with full shell and browser access when external tools are required.
  </Card>
  <Card
    title="AutoCheck"
    icon="circle-check"
    href="/features/autocheck">
    Automatically run pull-requests in multiple scenarios and fix any issues that are found.
  </Card>
</CardGroup>
