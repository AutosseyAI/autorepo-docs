---
title: 'Forks'
description: 'Use Autorepo to learn about a GitHub repository.'
---

You can use Autorepo to learn about GitHub repositories by forking them and creating an issue to ask Autorepo questions about the repository.

<Steps titleSize="h3">
  <Step title="Fork" stepNumber={1}>
    [Fork](https://docs.github.com/en/get-started/quickstart/fork-a-repo) the repository you want to learn about so that it is available on your GitHub account.
  </Step>
  <Step title="Create an Issue" stepNumber={2}>
    Create an issue in the repository you forked, asking Autorepo a question about the repository. Make sure to add the `auto` label so that Autorepo knows to respond.
  </Step>
  <Step title="Chat with Autorepo" stepNumber={3}>
    Ask follow-up questions to learn more!
  </Step>
</Steps>
